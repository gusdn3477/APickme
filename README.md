## APick Me (쉽고 간편한 채용 프로세스를 경험해 보세요)

프로젝트 기간 : 2021.10 - 2021.11

프론트엔드 : React.js  
백엔드 : Spring Boot, Spring Cloud Eureka, JPA, AWS RDS(Mariadb), AWS EC2, Docker,    
협업 툴: github, freedcamp
